Tecnologias para o desenvolvimento

É necessário a definição das tecnologias para que possamos ter uma boa manutenibilidade e que a busca de desenvolvedores seja fácil e não muito custosa.

  • C#

    A Linguagem de programação C# é orientada a objetos e desenvolvida pela Microsoft. A mesma será usada para o desenvolvimento do aplicativo desktop para Windows.

  • SQL Server

    O SQL Server é um SGBD desenvolvido pela Microsoft. O mesmo será utilizado para o armazenamento dos dados de toda aplicação, sendo alguns deles dados dos livros, leitor, fornecedor, empréstimo, reserva, bibliotecário, login, e mais alguns. A linguagem também será SQL Server para construção das Querys.

  • HTML

    O HTML é uma linguagem de marcação de texto usada para estruturação de páginas WEB. Por exemplo a que será acessada pelo leitor para consultar seus dados de empréstimo, reserva e livros.

  • CSS

    O CSS é usado para estilizar elementos HTML, ou seja, definir os espaçamentos, tamanhos, cores, fontes, fundos e etc.

  • JavaScript

    O JavaScript é uma linguagem de programação para WEB, é responsável por gerenciar eventos da página HTML. Por exemplo clique de um botão, população de uma tabela, etc.

  • PHP

    O PHP é uma linguagem de programação voltada para o back-end. A mesma faz integração com o banco de dados e recebe as requisições do HTML ou JavaScript. Por exemplo um formulário é enviado como POST e o PHP recebe os dados e registra no banco de dados.
